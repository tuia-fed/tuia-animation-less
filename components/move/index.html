# move 位移

# 介绍

  • 元素位移

# 代码演示

# 垂直移动

  .move_y();

# 水平移动

  .move_x();


# API

# 参数

.move_y(@from, @to, @duration, @timing-function, @delay, @iteration-count)

参数 说明 类型 默认值
from 动画开始位置 百分数或者带px的数字 100%
to 动画结束位置 百分数或者带px的数字 0
duration 动画完成所需时间 带s的number 0.3s
timing-function 动画速度曲线,可选值 linear、ease、ease-in、ease-in-out、step-start、step-end等 string ease
delay 动画延迟时间 number 0.1s
iteration-count 动画在结束前运行的次数,可选值 infinite或数值 number forwards
