Less动画库

# breathe 呼吸

# 介绍

  • 给元素添加呼吸动效

# 代码演示

# 默认

<div class="tuia_animate_animated tuia_animate_breathe"></div>

# less函数

  .breathe();

# 自定义动画完成所需时间

  .breathe(0.95, 0.7s);

# 自定义动画呼吸幅度

  .breathe(0.8, 0.7s);

# 自定义动画速度曲线

  .breathe(0.95, 0.7s, ease-in-out);

# @keyframes

animation: breathe;
animation-duration: 0.5s;
animation-timing-function: ease-in;
animation-iteration-count: infinite;

# API

# 参数

.breathe(@scaleNum, @duration, @timing-function, @iteration-count)

参数 说明 类型 默认值
scaleNum 呼吸缩放倍数 number 0.95
duration 动画完成所需时间 number 0.7s
timing-function 动画速度曲线,可选值 linear、ease、ease-in、ease-in-out、step-start、step-end等 string ease
iteration-count 动画在结束前运行的次数,可选值 infinite或数值 number infinite
