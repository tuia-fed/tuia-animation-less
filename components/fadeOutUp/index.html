Less动画库

# fadeOutUp 向上淡出

# 介绍

  • 常用于元素隐藏淡出动效，信息滚动消失时。

# 代码演示

# 基础

<div class="tuia_animate_animated tuia_animate_fadeOutUp"></div>

# less函数

  .fadeOutUp();

# less函数自定义

  .fadeOutUp(1s);

# @keyframes

animation-name: fadeOutUp;
animation-duration: 0.4s;
animation-fill-mode: forwards;

# API

# 参数

.fadeOutUp(@duration)

参数 说明 类型 默认值
duration 动画完成所需时间 number 0.2s
